# Mercado Livre Ads - Relatório Estratégico Automatizado

## Como rodar local
1. Instale dependências:
   pip install -r requirements.txt

2. Rode:
   streamlit run app.py

## Como usar
- Suba o Relatório de Campanhas do período
- Suba o Relatório de Anúncios Patrocinados do mesmo período
- Receba o Relatório Estratégico com:
  - Diagnóstico executivo
  - Matriz de oportunidade
  - Plano tático 7 dias
  - Painel geral com ação por campanha
  - Corte de sangria por anúncio

## Observação importante
Se o seu relatório de campanhas não tiver as colunas de Perda por orçamento e Perda por rank,
o app ainda roda, mas a classificação por quadrante fica menos precisa.
